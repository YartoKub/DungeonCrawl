## Readme журнал нововведений за 1 января - 31 июня 2026 года. 

#### 23 Апреля

Демонстрация ранее реализованного функционала по ссылке: https://rutube.ru/video/e5138e5c38e5b81e071ad5588877c246/

Показаны инструменты для работы с плоскими чанками и их полигонами, добавление, выделение, удаление, показаны два режима добавления полигона: монолитический и заполняющий. Демонстрация невозможности добавления внутреннего полигона в плоский чанк. Демонстрация работы закрытия дыр. Демонстрации не глубокие, без визуализации промежуточных действий.
 
#### 10 Марта

Использование графа связей полигонов для определения потенциальных дыр для их закрытия новым полигоном. 

Также детский эксперимент с системой сохранения файлов на компьютере и очень простая система чанков. Там есть бегунок, который загружает соседние чанки и может вносить изменения цвета в свой чанк. При удалении от чанков они отгружаются и сохраняются в файловой системе. 

#### 20 января

Готова рабочая система для Boolean операций над полигонами с маркировкой граней. Грани маркируются на Inside, Outside, и на Out_Colin и Inn_Colin для общих граней.

#### 11 января

Работа над алгоритмом пересечения полигонов с маркировкой граней.
